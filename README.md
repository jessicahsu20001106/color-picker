# Color Picker

Real-time iOS app for capturing real-world colors and converting them into HEX values.

<img width="410" height="292" alt="image" src="https://github.com/user-attachments/assets/6fadf47b-a559-454f-a2b2-abb7dc711203" />


## Features

- Capture colors directly from the surrounding environment using the camera
- Display detected colors as HEX values in real time
- Copy HEX codes with a single tap
- Save frequently used colors for quick access
- Minimal and interactive SwiftUI interface
- Home screen widget support with WidgetKit

## Tech Stack

- Swift
- SwiftUI
- AVFoundation
- WidgetKit
- Xcode

## Local Development

Clone the repository and open the project in Xcode:
```bash
git clone https://github.com/jessicahsu20001106/color-picker.git

open ColorPicker.xcodeproj
```

## Requirements

- iOS 17+
- Xcode 16+
